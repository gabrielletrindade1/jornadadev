# Exercício 1 - Função ou Procedimento?

## Conceito

- **Função:** executa uma operação e **retorna um valor**.
- **Procedimento:** apenas **executa uma ação**, sem retornar um valor.

---

### 1. `Str()`
**Descrição:** Converte um número em uma string.

**Resposta:** Função.

**Justificativa:** Recebe um número como parâmetro e retorna uma string correspondente ao seu valor.

---

### 2. `QOut()`
**Descrição:** Exibe um texto na tela.

**Resposta:** Procedimento.

**Justificativa:** Apenas exibe uma mensagem na tela, sem retornar um valor.

---

### 3. `Date()`
**Descrição:** Retorna a data atual.

**Resposta:** Função.

**Justificativa:** Retorna a data atual do sistema.

---

### 4. `Len()`
**Descrição:** Retorna o tamanho de uma string ou array.

**Resposta:** Função.

**Justificativa:** Retorna a quantidade de caracteres de uma string ou o número de elementos de um array.

---

### 5. `Upper()`
**Descrição:** Converte uma string para letras maiúsculas.

**Resposta:** Função.

**Justificativa:** Recebe uma string e retorna uma nova string com todos os caracteres em letras maiúsculas.

---

### 6. `ClearScreen()`
**Descrição:** Limpa a tela.

**Resposta:** Procedimento.

**Justificativa:** Executa a ação de limpar a tela, sem retornar nenhum valor.
